Question And Answer

Who does javascript work ?

The way JavaScript works is really very interesting. JavaScript basically runs on the client machine inside the browser. No compilation steps in JavaScript. JavaScript cannot read all the code at once. When a browser loads a page written in JavaScript, an interpreter in the browser reads the JavaScript code one line at a time and then runs it.

Session Storage VS Local Storage.

Session Storage: Session storage is where the data lasts as long as the browser is running and when the browser is closed the data will be gone. Later, when you log in to the browser again, session storage is used.

Local Storage: Local storage is when the browser stores the data while the browser is running, and when the browser is restarted, the browser shows the data that was previously stored.
